*크롤링 (Crawling)
: 인터넷에 있는 정보 중 우리가 원하는 것만 골라서 자동으로 수집해주는 기술

우리가 정보를 가져오려는 사이트를 불러와서 원하는 정보를 찾고 가져옴

step of text/image Crawling

1. 개발환경 설정
-> 버전끼리 충동할 수 있어서 보통 가상환경을 구성한다
(venv, virtualenv, conda...)
python -m venv myvenv

*강의에서는 <구름IDE> 개발환경이 갖춰진 가상의 컴퓨터를 제공

2. 예제 무작정 실행
*Beautiful Soup
: HTML 및 XML 구문을 분석하기 위한 Crawling용 파이썬 패키지 (라이브러리)
-> 명령어로 bs4 모듈 설치하자
pip (패키지 관리자)


3. 다른 작업에 응용 (네이버 실검 text 저장)

4. 라이브러리 사용 -> 동물상 연예인 Crawling
